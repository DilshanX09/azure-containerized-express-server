This is the home page. Use the API endpoints to interact with the server.

Available API Endpoints

Latest Updated: 02/11/2026

GET   /api/users - Retrieve a list of all users.

POST /api/users - Create a new user. (Requires JSON body with user details)

GET /api/users/:id - Retrieve details of a specific user by ID.

PUT /api/users/:id - Update details of a specific user by ID. (Requires JSON body with updated user details)

DELETE /api/users/:id - Delete a specific user by ID.

GET /health - Check the health status of the server.
